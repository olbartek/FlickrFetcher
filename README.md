# FlickrFetcher
An application that fetches photos from Flickr based on a given tag.

## Used technologies
- Swift 3.0
- Protocol Oriented Programming (POP)
- Dependecy injection
- API Pagination
- OperationQueue (used for cancelling operations when needed)
- Realm (used for caching)
